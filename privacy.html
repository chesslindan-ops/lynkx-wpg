Privacy Policy

1. What Data LynkX Processes

To function, LynkX may process:

From Discord:
• Message content only when required to detect links
• User IDs (for command usage and permissions)
• Guild IDs and channel IDs
• Settings configured by server owners/admins

LynkX does NOT collect or store:
• Direct messages
• Passwords
• Emails
• IP addresses
• Sensitive personal information

2. How Data Is Used

LynkX uses data only to:

• Detect and format links
• Provide bot features
• Maintain per-server settings
• Respond to commands

No data is shared, sold, or transmitted to third parties.

3. Data Storage

The bot may store:

• Detected links
• Server settings
• Timestamps
• User IDs tied to bot interactions

Data is stored only as long as needed for the bot’s features.

4. Data Removal

Server owners may request deletion of all stored data related to their server.
