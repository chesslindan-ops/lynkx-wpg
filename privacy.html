Privacy Policy

1. What Data LynkX Processes

To function, LynkX may process: message content only when required to detect links; user IDs for command usage and permissions; guild IDs and channel IDs; and settings configured by server owners/admins. LynkX does not collect or store direct messages, passwords, emails, IP addresses, or sensitive personal information.

2. How Data Is Used

LynkX uses data only to detect and format links, provide bot features, maintain per-server settings, and respond to commands. No data is shared, sold, or transmitted to third parties.

3. Data Storage

The bot may store detected links, server settings, timestamps, and user IDs tied to bot interactions. Data is stored only as long as needed for the bot’s features.
