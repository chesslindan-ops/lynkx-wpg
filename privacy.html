Privacy Policy

How LynkX handles your data, what is collected, and how to request deletion.

1. Data LynkX Processes

  • From Discord:
    • Message content when required to detect links
    • User IDs for commands and permissions
    • Guild & channel IDs
    • Server settings
  • Not collected:
    • Direct messages
    • Passwords
    • Emails
    • IP addresses
    • Sensitive personal information

2. How Data Is Used

  • Detect and format links
  • Provide bot features
  • Maintain per-server settings
  • Respond to commands

No data is shared or sold to third parties.

3. Data Storage

  • Detected links
  • Server settings
  • Timestamps
  • User IDs tied to interactions

Stored only as long as needed for bot features.

4. Data Removal

Server owners can request deletion of all data related to their server.
